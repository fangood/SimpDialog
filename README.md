# SimpDialog
基本样式的Dialog

1、单个button的dialog
2、两个button的dialog
3、带列表的dialog
4、带输入框的dialog
5、自定义消息标题、消息内容

-----------------------------------------

Uses:

Step 1. Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}Copy
Step 2. Add the dependency

	dependencies {
	        implementation 'com.github.fangood:SimpDialog:Tag'
	}
